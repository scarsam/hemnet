# README

About: New rails project with Webpacker and Typescript, fetching data from the movie db API, exposing the data to the client through GraphQL query.

Libraries used:

- GraphQL
- React / Typescript
- Formik
- Apollo
- Yup
- React Testing Library
- TailwindCSS
- Rspec / VCR

API used:

- https://developers.themoviedb.org/3
